CoreManager for ArcEmu, MaNGOS, and Trinity Core Servers


ABOUT:

   CoreManager is a PHP, web front end for Multiple Cores that
 provides easy access to managing MMORPG servers. Manage/add/remove/edit/
 mute/ban/etc. user accounts, characters, database and server itself.
 It is a small, easy to use, flexible tool.

   CoreManager uses data on databases provided by the server core, but it also
 requires additional data provided in /SQL/ folder.

   The user Security Level is NOT based on the accounts' gm level from the server core.
 Highest access level, by CoreManager default is 4 (SysOp).
 Default page configuration allows players to view most pages, all as read-only,
 while SysOp level has full access.
 
   If you like CoreManager, add yourself as a user on Ohloh. (https://www.ohloh.net/p/CoreManager)
